# Introduction

前端开发应用领域广泛：web开发、移动web开发、APP开发、小程序开发、服务端开发等

## 前端开发主要技术介绍

| 技术 | 介绍 |
| :--- | :--- |
| HTML5 | 搭建网页结构的语言 |
| CSS3 | 层叠演示表，美化网页的语言 |
| JavaScript | 增加网页互动功能的语言 |
| Vue.js/React.js | 前端框架，采用模块化开发、数据驱动、声明式编程 |

|        | 语言       | 功能                             |
| ------ | ---------- | -------------------------------- |
| 结构层 | HTML       | 搭建结构、放置部件、描述语义     |
| 样式层 | CSS        | 美化页面、实现布局               |
| 行为层 | JavaScript | 实现交互效果、数据收发、表单验证 |
